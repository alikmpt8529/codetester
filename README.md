# Coding Rule Checker

C言語コーディング規約チェッカー - クロスプラットフォーム対応デスクトップアプリケーション

## 概要

このアプリケーションは、C言語のソースコードがコーディング規約に準拠しているかをチェックするツールです。Mac と Windows の両方で動作し、直感的なユーザーインターフェースを提供します。

## 主な機能

- **コーディング規約ファイルのアップロード**: 主要規約と二次規約の両方に対応
- **Cソースファイルの解析**: 規約違反の自動検出
- **詳細なレポート生成**: 違反箇所と修正提案を含むレポート
- **修正版ファイルの生成**: 違反箇所にコメントを追加したファイルの出力
- **課題生成機能**: 教育用のC言語課題ファイル生成
- **クロスプラットフォーム対応**: macOS と Windows で動作

## 開発環境のセットアップ

## 使用方法

1. **規約ファイルのアップロード**
   - 主要コーディング規約ファイル（.txt）をアップロード
   - 必要に応じて二次規約ファイルもアップロード

2. **ソースファイルのアップロード**
   - チェック対象のCソースファイル（.c）をアップロード

3. **解析の実行**
   - 「規約チェック実行」ボタンをクリック
   - 解析結果の確認

4. **結果の確認と出力**
   - 違反レポートの確認
   - 修正版ファイルのダウンロード
