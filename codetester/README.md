# 仕様
全体仕様
    このプロジェクトをgitにアップし、このファイルが仕様であることを明記する
//以下はこのアプリの仕様です
1.コーディング規約を.txtファイルで指定するので、そのファイルを読み込む
2.追加の規約があればその都度.txtファイルで指定するので、そのファイルも読み込む
3..cファイルをアップロードすると、そのファイルが指定されたコーディング規約（１、２でアップロードしたもの）に準拠しているかどうかをチェックする
4.もし規約に準拠していなければ、どの部分が規約のどの部分に違反しているかを.txtファイルで出力する
4.1 規約に準拠していればcorrectと返り値を与える
